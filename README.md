# Sibiya AI

**Umoya Wesizwe SakwaZulu**

A personal AI assistant built for the Sibiya family and ventures — Sim Wear, DevMzansi, and whatever comes next. Talks back in a real voice, reads isiZulu fluently, and reads your documents.

---

## About

Sibiya AI isn't a generic chatbot. It's trained on real context — Sim Wear, DevMzansi, and the other projects that make up the Sibiya name — so every answer comes from an assistant that actually knows what it's talking about.

Built by [Sizwe Sicelimilo Sibiya](https://github.com/Izzy-10), a frontend developer student at WeThinkCode_ Durban.

## Features

- **Talks back** — every reply can be read aloud in a male voice, tap-to-play per message
- **isiZulu fluent** — replies fully in isiZulu if you write in isiZulu, not just scattered phrases
- **Reads PDFs** — attach a document and ask questions about it directly
- **Knows the ventures** — trained on Sim Wear, DevMzansi, KickOff Intelligence, SmartOnboard, and README Doctor
- **Dark + gold identity** — consistent design language across every Sibiya project

## Design

- **Palette:** black (`#0B0B0C`) and gold (`#C9A44C`)
- **Type:** Space Grotesk for headers, system sans for body
- **Logo:** angular "S" monogram, framed with an amasumpa beadwork pattern — a nod to traditional Nguni design, not a generic tribal cliché

## Tech Stack

- React (single-file artifact)
- Anthropic API (Claude Sonnet)
- Web Speech API (voice output)
- Tailwind utility classes
